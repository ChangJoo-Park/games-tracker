# Games Tracker

## Dependencies

- Ruby 2.5.1
- Node 9.11.1
- PostgreSQL 10.4

## Development

```sh
$ brew install hivemind
$ bundle install
$ yarn install
$ yarn setup
$ yarn start
```

## Test

To start api tests run:

```sh
$ yarn test:api
```

To start client tests run:

```sh
$ yarn test:client
```

To start end-to-end tests run:

```sh
$ yarn cypress
```
